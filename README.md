# ML Classification Experiments (Python)

This repository demonstrates a complete machine learning
classification workflow using classical ML algorithms.

## Features
- Data preprocessing and feature selection
- Train/test split
- Multiple models:
  - Decision Tree
  - KNN
  - Support Vector Machine (SVM)
  - Random Forest
  - Multi-Layer Perceptron (MLP)
- Model comparison using test accuracy and cross-validation
- Confusion matrix and classification report

## Technologies
- Python
- scikit-learn
- pandas
- numpy
- matplotlib
- seaborn

## Notes
The dataset used in this project is not included for privacy reasons.
You can replace it with your own CSV file, following the same structure.
